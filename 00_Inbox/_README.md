# 📥 00_Inbox — 缓冲区

> 所有新想法、临时笔记、待整理内容先放在这里。

## 使用规则

- **快速捕获**：任何灵感、想法、会议记录先扔这里，不要纠结分类
- **定期清理**：每周抽时间把笔记归档到 `10_Space` 或 `20_Common`
- **AI 提示**：AI 可以帮助自动分类和归档 Inbox 中的内容

## 命名建议

```
YYYY-MM-DD_简短描述.md
```

## 状态标签

- `#inbox/new` — 刚扔进来的
- `#inbox/processing` — 正在整理
- `#inbox/done` — 已归档，可删除
