# 🚀 10_Space — 主工作区

> 按项目/领域组织的核心工作区，存放 MD、PDF、Excel 等各类资料。

## 目录组织原则

每个子文件夹代表一个**项目**或**领域**：

```
10_Space/
├── 海克斯大乱斗/       # 游戏策略研究
├── uc对局复盘/         # 对局数据分析
├── 项目名称/           # 你的其他项目...
│   ├── _README.md      # 项目概述（推荐）
│   ├── notes/          # 笔记文档
│   ├── data/           # 数据文件
│   └── assets/         # 图片等资源
```

## 命名建议

- 文件夹名用中文或英文均可，保持简洁
- 每个项目建议放一个 `_README.md` 说明项目背景
- 数据文件（xlsx, csv）放在项目子目录下

## AI 约定

- AI 在此目录下工作时，应先阅读项目的 `_README.md`
- 生成的分析报告放在对应项目目录下
