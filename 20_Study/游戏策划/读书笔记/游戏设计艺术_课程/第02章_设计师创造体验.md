# 第2章 设计师创造体验

> **板块**：基础理念 | **课时**：02/34 | **对应透镜**：#1 情感、#2 本质体验
> **一句话总结**：游戏只是手段，创造令人难忘的**玩家体验**才是目的。

---

## 📖 本章概述

如果第1章回答了"你是谁"，第2章就回答了"你在做什么"。答案不是"做游戏"——而是"**创造体验**"。这个区别至关重要：游戏是你能控制的东西（代码、规则、资源），但体验是发生在玩家内心的东西（情感、记忆、感受）。你永远无法直接创造体验，只能创造**产生体验的条件**。

---

## 🔑 核心概念

### 2.1 游戏 ≠ 体验

Schell 用了一个精妙的比喻：

> **游戏设计就像制作瓶中船。** 你不能直接在瓶子里组装船——你在外面精心制作船的零件，巧妙地放进瓶子里，用工具远距离操控，让它在瓶内展开。游戏设计也是如此：我们被远远隔离在想要创作的体验之外，只能创造人工元素让玩家交互，祈祷玩家能从这些交互中获得我们期望的体验。

**这意味着什么？**

```
设计师创造的          →    玩家获得的
─────────────          ─────────────
游戏规则               →    理解/困惑
关卡设计               →    探索/成就
角色对话               →    共情/感动
难度曲线               →    心流/挫败
视觉效果               →    沉浸/惊叹
```

设计师和体验之间总是隔着一层。你设计的是**左边**的东西，但你真正关心的是**右边**的东西。

### 2.2 体验是独一无二的

每个玩家的体验都是独特的，因为体验由三者共同决定：

```
            ┌─────────────┐
            │   游戏本身    │ ← 你能控制的
            └──────┬──────┘
                   │
                   ▼
         ┌─────────────────┐
         │   玩家的内心世界   │ ← 你不能控制的
         │ （认知、情感、记忆、│
         │   性格、文化背景）  │
         └────────┬────────┘
                  │
                  ▼
           ┌──────────────┐
           │  具体的体验    │ ← 你真正关心的
           └──────────────┘
```

同一款游戏：
- 一个老玩家感到"太简单了"
- 一个新手感到"太难了"
- 一个故事爱好者被剧情感动流泪
- 一个竞技玩家完全跳过了剧情

**你能做的**：尽可能理解你的目标玩家，设计出最可能产生预期体验的游戏。

### 2.3 三个帮手：心理学、人类学、设计学

为了缩小"设计意图"和"实际体验"之间的鸿沟，Schell 引荐了三个学科作为设计师的帮手：

#### 🧠 心理学（Psychology）
- **作用**：理解人类认知和情感的运作机制
- **关键概念**：
  - 注意力如何分配？（为什么玩家会忽略教程？）
  - 记忆如何工作？（为什么峰值体验比持续体验更重要？）
  - 情感如何被触发？（为什么"差点死掉"比"轻松通关"更令人兴奋？）
  - 动机如何形成？（为什么有些奖励反而降低了动力？）
- **推荐子领域**：认知心理学、行为心理学、积极心理学

#### 🌍 人类学（Anthropology）
- **作用**：将文化研究方法引入玩家研究
- **关键概念**：
  - 参与式观察——融入玩家社群，观察他们的自然行为
  - 文化差异——不同国家、地区的玩家有不同的审美和价值观
  - 仪式与象征——游戏中的"仪式感"（如开箱、升级动画）为什么让人着迷
- **实践方法**：
  - 不是给玩家发问卷，而是坐在他们旁边看他们玩
  - 加入玩家的 Discord/论坛，像"原住民"一样生活
  - 观察玩家发明的"非官方玩法"——那才是他们真正想要的

#### 🎨 设计学（Design）
- **作用**：从所有创造体验的领域学习
- **关键概念**：
  - 工业设计——如何让物体使用起来"自然而然"
  - 建筑设计——如何用空间引导人的行为
  - 平面设计——如何用视觉层次传达信息
  - 体验设计——主题公园、博物馆、商店是如何设计"体验流"的
- **Schell 的忠告**：不要只从游戏行业内部寻找灵感，最好的创意往往来自跨界

### 2.4 海森堡原理与体验分析

Schell 提出了一个有趣的悖论：

> **分析体验会破坏体验。** 就像量子力学中的海森堡不确定性原理——观测行为本身会改变被观测的对象。当你试图在体验的同时分析它，你就会"跳出"体验，变成一个旁观者。

这是设计师面临的独特挑战：**你需要深入理解体验，但深入分析会破坏体验本身。**

#### 三种"战胜"海森堡原理的方法

**① 分析回忆（Analyze Memories）**
- 不要在体验过程中分析，事后再回忆和分析
- 记忆虽然不完美，但能保留体验的**情感轮廓**
- 练习：玩完一个游戏后，立刻写下你记住的5个最强烈的时刻

**② 经历两次（Experience Twice）**
- 第一次：完全沉浸，不做任何分析
- 第二次：带着分析的目光重新体验
- 第一次给你"真实感受"，第二次给你"设计洞察"
- 这就是为什么好的设计师会反复玩同一个游戏

**③ 默默观察（Silently Observe）**
- 培养一种"分裂"的能力——同时是参与者和观察者
- 像冥想一样，保持一种"觉知"的状态
- 这需要长期练习，但一旦掌握，会极大提升你的设计敏感度

### 2.5 本质体验

Schell 提出了一个核心概念——**本质体验（Essential Experience）**：

> 每一种体验都有一个"本质"——剥离所有装饰后留下的核心感受。你的游戏不需要完美复制某种体验的每个细节，只需要**抓住它的本质**。

**案例解析**：

| 你想传达的体验 | 不需要 | 需要抓住的本质 |
|--------------|--------|-------------|
| 战争的残酷 | 真实的子弹物理模拟 | 失去同伴的痛苦、每个决定都有代价 |
| 探险的刺激 | 100%写实的地理 | 未知感、发现的惊喜、危险与奖励的平衡 |
| 友谊的温暖 | 复杂的社交系统 | 共同经历困难后的情感连接 |
| 速度的快感 | 真实的汽车引擎模型 | 风驰电掣的视觉冲击、"差一点就撞了"的紧张感 |

> 💡 **关键洞察**：本质体验通常不是"事实"层面的，而是"情感"层面的。《马力欧赛车》不需要真实的物理引擎——它需要的是"你追我赶、险象环生"的**感觉**。

### 2.6 你的感受都是真实的

Schell 强调了一个容易被忽视的点：

> **在游戏中产生的情感是真实的。** 玩家在游戏中感到的恐惧、愤怒、快乐、悲伤——这些不是"假"的情感。大脑产生的化学反应是真实的。

这意味着：
1. 不要轻视游戏体验——"这只是个游戏"是一种误解
2. 你有能力创造真正影响人的情感体验
3. 这也带来了设计师的**责任**——你创造的体验会真实地影响玩家

## 📌 本章透镜

### 🔍 透镜#1：情感透镜（The Lens of Emotion）

> **问自己这些问题：**
> 1. 我希望玩家体验到怎样的情感？为什么？
> 2. 这些情感在游戏中是如何被触发的？
> 3. 玩家的实际情感反应是否与我的预期一致？
> 4. 如果不一致，如何调整设计来接近预期？

**使用指南**：
- 在设计的**每个阶段**都使用这个透镜
- 情感不是"加上去"的——它应该是设计的**出发点**
- 不要试图同时传达太多情感——选择1-3种核心情感，全力设计

**示例应用**：
- 《Journey（风之旅人）》：核心情感 → 孤独、渺小、归属
- 《Dark Souls》：核心情感 → 恐惧、成就、发现
- 《动物森友会》：核心情感 → 平和、归属、创造的喜悦

---

### 🔍 透镜#2：本质体验透镜（The Lens of Essential Experience）

> **问自己这些问题：**
> 1. 我想让玩家获得怎样的体验？
> 2. 这种体验的**本质**是什么？
> 3. 我的游戏如何抓住这个本质？
> 4. 游戏中有哪些元素在削弱这个本质？应该去掉吗？

**使用指南**：
- 用一个词或一句话描述你的本质体验
- 游戏中的每个元素都应该服务于这个本质
- 如果某个元素很酷但与本质体验无关，要有勇气删掉它

## 🎮 案例分析

### 案例1：《Journey（风之旅人）》的体验设计

陈星汉（Jenova Chen）在设计《风之旅人》时，明确定义了他想要创造的本质体验：**"在一个陌生的世界中，两个素不相识的旅人之间产生真诚的情感连接。"**

为了抓住这个本质：
- 去掉了语音聊天（言语会破坏纯粹的情感）
- 去掉了竞争元素（竞争会取代合作）
- 只保留简单的"发出音符"互动（迫使玩家用行动而非语言交流）
- 设计了共同面对困难的场景（患难中建立连接）

### 案例2：恐怖游戏的"分析回忆"

一个恐怖游戏设计师的工作流程：
1. 第一次玩《P.T.》——在黑暗中被吓得心跳加速
2. 事后回忆——记住了"走廊变化"的恐惧、"不知道下一步会发生什么"的焦虑
3. 第二次玩——注意到灯光变化的节奏、音效的微妙递进、空间重复但微变的设计技巧
4. 提炼本质——恐惧不来自怪物本身，而来自"熟悉环境中的微妙变化"

## ✏️ 思考与练习

1. **情感地图**：选一个你最近玩的游戏，画一条"情感曲线"——随时间变化，你的情感强度和类型是怎样的？哪些时刻最强烈？

2. **本质提炼**：想象你要设计一个"登山"主题的游戏。写下这种体验的"本质"是什么——不是"登山"的物理过程，而是它的**情感核心**。

3. **海森堡练习**：今天玩一个游戏，用"分析回忆"法——玩完后立刻关掉游戏，闭眼回忆你最强烈的5个感受，写下来。

4. **跨界本质**：看一部电影，提炼它的"本质体验"。然后思考——如果把这个本质体验做成游戏，你会怎么设计？
