# 📖 《快乐之道：游戏设计的黄金法则》课程

> **原书**：*A Theory of Fun for Game Design*（第2版，10周年修订版）
> **作者**：Raph Koster（拉夫·科斯特）
> **出版**：O'Reilly Media，全彩印刷
> **核心论点**：**快乐 = 学习。** 游戏的乐趣本质上是大脑在识别、学习和掌握模式的过程中获得的奖赏。

---

## 📚 课程导读

### 这本书的独特之处

《快乐之道》是游戏设计领域最经典的理论著作之一，被全球数十个大学的游戏设计课程采用。它有两个极为独特的特点：

1. **左文右图**：每一个对开页，左页是理论文字，右页是 Koster 亲手绘制的幽默插图。图文互补，让深奥的理论变得直观可感
2. **跨学科视角**：从认知科学、进化心理学、神经科学、美学理论等多个角度解构"乐趣"的本质

### 作者背景

Raph Koster 是 MMORPG 先驱：
- 《网络创世纪》（Ultima Online）首席设计师——开创了现代 MMO 设计范式
- 《星球大战：星系》（Star Wars Galaxies）创意总监
- 2012年 GDC Online "在线游戏传奇"荣誉获得者
- 从 MUD 时代到社交游戏，横跨游戏产业几乎所有领域

### 学习建议

- ⏱️ **预计学习时间**：12-15 小时（每章约1小时）
- 📝 **学习方式**：建议先快速通读一遍，再逐章精读
- 🎮 **实践方法**：每章结束后，用文中理论分析一款你熟悉的游戏
- 🔗 **配合阅读**：可与 [[游戏设计艺术_课程/_README|《游戏设计艺术》课程]] 交叉参考

## 📊 全书核心公式

```
快乐 = 学习 = 大脑成功识别模式时的多巴胺奖赏

无聊 = 模式被完全掌握（图标化） = 没有新东西可学

挫败 = 模式太复杂，无法识别 = 面对噪音

好游戏 = 持续提供"可学习的新模式" = 足够深广的"可能性空间"
```
