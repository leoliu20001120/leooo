# 📖 20_Study — 学习成长区

> 个人技术学习、读书笔记、课程记录等与自我提升相关的内容。

## 目录结构

```
20_Study/
├── 技术栈/               # 编程语言、框架、工具的学习笔记
│   ├── Python/
│   ├── Go/
│   └── ...
├── 读书笔记/             # 书摘和阅读总结
│   ├── 《书名》.md
│   └── ...
└── 课程笔记/             # 在线课程、培训、认证
    ├── 课程名称/
    └── ...
```

## 与 10_Work 的区分

| 放这里 ✅ | 放 10_Work ❌ |
|-----------|--------------|
| 技术学习笔记 | 具体工作项目文档 |
| 读书摘要、书评 | 业务数据分析 |
| 课程/教程笔记 | PRD、方案设计 |
| 编程练习、Demo | 会议纪要、周报 |
| 认证考试准备 | 项目复盘报告 |

## 进阶用法

- 学习过程中发现的**通用方法论**，可提炼后归档到 `30_Common/方法论/`
- 与工作项目相关的技术调研，放在对应的 `10_Work/项目/` 下
- 用 `[[双链]]` 关联学习笔记和工作实践，形成知识网络

## 命名建议

```
技术栈/    → 语言名_主题.md      如：Python_装饰器.md
读书笔记/  → 《书名》.md         如：《设计模式》.md
课程笔记/  → 课程名/章节.md      如：MIT6.824/Lab1.md
```

## AI 约定

- AI 可以帮助整理学习笔记、生成知识卡片
- 推荐用 Markdown + 代码块记录技术笔记
- 生成的学习总结放在对应子目录下
