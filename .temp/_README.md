# 🔧 .temp — AI 后厨

> 这是 AI 的工作目录，用于存放中间数据、临时脚本和缓存索引。
> 人类通常不需要关注此目录。

## 子目录说明

| 目录 | 用途 | 示例 |
|------|------|------|
| `data/` | MCP 抓取的数据或脚本产生的中间数据 | API 返回的 JSON、清洗后的 CSV |
| `scripts/` | 临时运行的 Python/JS 脚本 | 数据分析脚本、批量处理脚本 |
| `cache/` | 影子索引，加速 AI 检索 | 全库摘要、标签索引、关系图 |

## 清理策略

- `data/` 和 `scripts/` 中超过 30 天的文件可安全删除
- `cache/` 会被 AI 按需重建，删除无风险
