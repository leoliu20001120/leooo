# 海克斯大乱斗 AI 教练设计方案

## 一、背景与目标

### 核心洞察
- 新手玩家玩海斗的留存显著高于其他模式，但主动选择海斗的人并不多
- 推测原因之一：前置教程和引导不足，新手虽适合但没被有效带进去
- 一局内要快速认识 10 个陌生英雄，理解成本高，容易劝退

### AI 教练的定位
在对局过程中做**轻量、及时、非强制**的教学伴侣，帮助新手：
1. 降低上手认知门槛（地图机制、英雄能力、海克斯选择）
2. 在关键决策点给出可操作的建议
3. 不打断游戏节奏，自然穿插在"教练没话聊"的空档

## 二、教学内容设计

### 模块 A：地图机制引导（乱斗 + 海斗通用）

#### A1. 传送门机制介绍
- **触发时机**：对局开始，玩家走路上线时
- **优先级**：低（"没话聊"时穿插）
- **话术示例**：

> "看到脚下发光的传送门了吗？踩上去就能直接飞到前线，比走路快多了。对面也能用你家的传送门，留意别被偷袭。"

> "外塔被拆之后，传送门会自动连到水晶防御塔。可以拿来搞侧翼包抄，也要小心对面绕后。"

#### A2. 三张地图的回复 / 特色机制

**嚎哭深渊（经典冰雪地图）**

| 机制 | 教练话术 |
|------|----------|
| 血包刷新 | "路上有红十字血包，吃到会回血回蓝，而且血越少回得越多。吃的时候还会撒一圈回血圈，队友站旁边也能蹭到。" |
| 泉水不回血 | "这张图泉水不回血哦，别指望回家续命，死了才能买装备。" |
| 家园起始加速 | "开局五分钟内出门会有加速buff，趁着加速赶紧到位。" |

**屠夫之桥（比尔吉沃特海盗风格）**

| 机制 | 教练话术 |
|------|----------|
| 地形布局 | "这张图是海盗港口风格，机制和嚎哭深渊一样，但地形视觉不同，别看花眼就行。" |

**进步之桥（双城之战主题·限时）**

| 机制 | 教练话术 |
|------|----------|
| 不对称血包 | "这张图两边血包效果不一样——皮城那边吃了能刷新技能CD，祖安那边吃了加攻击力和血量。记得多吃。" |
| 侧道 | "两边多了侧道，可以绕行包抄，也要小心被人蹲。" |
| 超级兵水晶加成 | "推掉水晶后全队会获得强力增益，其中一个人还会拿到超级buff，后期一定要抢着推。" |
| 雪球升级 | "雪球砸中后地上会出加速路径，队友踩上去也能加速冲过去，适合配合开团。" |

#### A3. 防御塔废墟
- **触发时机**：己方或敌方外塔被推掉时
- **话术示例**：

> "塔倒了之后碎石会挡住一边的路，可以利用这个地形卡位，也可能被对面堵住，注意走位。"

### 模块 B：海克斯强化符文教学（海斗专属）

#### B1. 选择时机提醒
- **触发时机**：首次进入符文选择界面时
- **话术示例**：

> "海克斯符文来了！三选一，选完这局就一直生效。后面 7 级、11 级、15 级还各有一次机会，别急，先看看哪个合适。"

#### B2. 海克斯标签体系

为每个海克斯强化符文打上多维标签，用于驱动 AI 教练的推荐逻辑和新手理解辅助。

##### 维度一：效果类型（这个符文"做什么"）

描述符文的核心机制作用，帮新手秒懂"选了之后我会变成什么样"。

| 标签 | 含义 | 典型符文举例 |
|------|------|-------------|
| 🗡️ 输出增幅 | 直接提升伤害数值（攻击力、法强、穿透等） | 大力、超凡邪恶、玻璃大炮 |
| 🔁 技能加速 | 缩短技能冷却，提高技能释放频率 | 循环往复、无限循环往复、终极刷新 |
| 🛡️ 生存增强 | 提升坦度（血量、护甲、魔抗、护盾） | 坦克引擎、歌利亚巨人、星界躯体 |
| ❤️‍🩹 续航回复 | 提供持续治疗、吸血、回城能力 | 吸血习性、无休回复、作弊我能回城 |
| 💨 机动性 | 提升移速、提供位移或闪现类能力 | 速度恶魔、逃跑计划、利刃华尔兹 |
| 🎯 攻击特效 | 为普攻或技能附加额外触发效果 | 虚幻武器、台风、接二连三、火上浇油 |
| 💥 爆发 | 在短时间内制造大量伤害的能力 | 巨人杀手、不祥契约、易损 |
| 🧲 控制/干扰 | 提供减速、击退、缩小等控场效果 | 缩小引擎、冰寒、闪光弹 |
| 🤝 团队增益 | 增强队友或提供团队范围效果 | 风语者的祝福、天音爆、午夜快递 |
| 🎲 特殊机制 | 改变基本游戏规则（回城、召唤师技能替换等） | 作弊我能回城、史上最大雪球、雪球扭蛋机 |
| 💰 经济收益 | 提供额外金币或资源获取能力 | 红包、捐赠、夺金 |

##### 维度二：适用英雄类型（"谁拿最好"）

基于英雄的战斗定位做匹配，帮教练快速判断"你这个英雄适不适合拿"。

| 标签 | 适用谁 | 判断依据 |
|------|--------|----------|
| 近战适用 | 战士、刺客、坦克等贴身英雄 | 需要近身才能触发、或近战收益远高于远程 |
| 远程适用 | 射手、法师、部分辅助 | 需要保持距离输出才能充分发挥 |
| 通用 | 所有英雄都有收益 | 效果不依赖射程或战斗方式 |
| 法术英雄优先 | 主要靠技能打伤害的英雄 | 效果与法强、技能释放频率挂钩 |
| 物理英雄优先 | 主要靠普攻打伤害的英雄 | 效果与攻速、暴击、攻击力挂钩 |
| 坦克/前排优先 | 需要站前面吸收伤害的英雄 | 效果与生命值、护甲、魔抗联动 |
| 辅助/增益优先 | 提供治疗、护盾的英雄 | 效果与治疗护盾强度联动 |

##### 维度三：强度评级（"值不值得拿"）

帮教练做"推荐优先级"判断，新手最需要的是明确告知"这个该不该选"。

| 标签 | 含义 | 教练表达方式 |
|------|------|-------------|
| S · 天胡必拿 | 无脑选，效果碾压同级 | "碰到就拿，别犹豫" |
| A · 强力推荐 | 大多数情况下都是好选择 | "这个很稳，选它不亏" |
| B · 看情况 | 契合特定英雄/阵容时很强 | "配你这个英雄挺合适" |
| C · 一般 | 有收益但不突出 | "没更好的可以先拿着" |
| D · 不推荐 | 收益低或条件苛刻 | 教练不主动推荐，仅在被问时解释 |

> **注意**：强度评级需要按版本跟随平衡性调整动态更新，不能写死。

##### 维度三补充：阶段化评级偏移

同一符文在不同选择阶段（开局/7级/11级/15级）的评级可能不同。数据上按 `hero × rune × stage` 三维度统计阶段胜率，评级在基础等级上做阶段偏移：

| 阶段 | 偏移规则 | 教练策略 |
|------|----------|----------|
| **开局（Lv.1）** | 简单易懂 + 不需配合的符文评级 +0.5 级；高阶玩法符文 -0.5 级 | 推万金油，黑科技仅提示 |
| **7级** | 与已有符文有黑科技联动的评级 +1 级 | 组合成型优先 |
| **11级** | 与已有装备/符文联动强的评级 +0.5 级 | 基于现有 build 推荐 |
| **15级** | 全部评级保守化（S→S, A→A, B→B+偏保守话术） | 不鼓励刷新，选最稳的 |

##### 维度四：品质等级（系统自带属性）

| 标签 | 含义 |
|------|------|
| 银色 | 基础强度，效果相对温和 |
| 金色 | 中等强度，通常有明显的玩法改变 |
| 棱彩 | 最高强度，往往能重新定义一个英雄的打法 |

##### 维度五：所属套装（26.3+ 版本新增）

标记符文归属哪个主题套装，用于触发"凑套装"提醒。

| 标签 | 套装效果方向 | 教练提示 |
|------|-------------|----------|
| 金币雨 | 经济加成 | "凑齐能额外赚钱" |
| 大法师 | 技能冷却返还 | "凑齐技能转更快" |
| 俯冲炸弹 | 复活 / 阵亡效果 | "凑齐死了还能炸一波" |
| 自动化 | 自动触发类效果 | "凑齐效果自动帮你打" |
| （其他套装） | 按版本扩展 | 动态填充 |

##### 维度六：装备联动（"和什么装备配最好"）

标记符文与特定装备 / 出装路线的增益关系，教练可在推荐符文后顺带提一句出装建议。

| 标签 | 联动关系 | 教练话术举例 |
|------|----------|-------------|
| 暴击联动 | 与无尽之刃、暴击装组合收益高 | "选了这个，装备往暴击方向出就对了" |
| 攻速联动 | 与攻速装（飓风、电刀等）形成循环 | "配上攻速装，攻击特效触发特别频繁" |
| 法强联动 | 与法强装（帽子、大面具等）倍增 | "法强堆越高，这个符文效果越猛" |
| 生命值联动 | 与血量装（狂徒、冰杖等）放大 | "出肉装的时候这个符文收益最大" |
| 穿透联动 | 与穿甲/法穿装互相补强 | "配合穿透装，对面护甲等于白出" |
| 吸血联动 | 与吸血装/全能吸血形成续航闭环 | "再出个吸血装，基本打不死你" |
| 特定装备 | 与某件具体装备有特殊互动 | "这个符文配纳什之牙特别猛" |

##### 维度七：对局阶段偏向（"早期选还是后期选"）

不同等级解锁节点适合选不同类型的符文，帮教练在不同阶段给出差异化建议。

| 标签 | 含义 | 说明 |
|------|------|------|
| 前期优选 | 开局 / 7 级时最大化收益 | 早期就能立刻发挥作用，不需要装备配合 |
| 中期优选 | 11 级左右收益最佳 | 需要一定装备基础才能发挥 |
| 后期优选 | 15 级时用于补强短板或决胜 | 适合收尾、补缺，或后期缩放好 |
| 全程通用 | 任何阶段选都不亏 | 收益与阶段无关，稳定保底 |

##### 维度八：决策难度（"新手能不能驾驭"）

控制教练推荐时是否需要额外解释，或是否应该优先推给新手。

| 标签 | 含义 | 教练策略 |
|------|------|----------|
| 🟢 简单易懂 | 选了就有效果，不需要额外操作 | 优先推荐给新手 |
| 🟡 需要配合 | 需要搭配特定装备或打法才能发挥 | 推荐时附带一句配合建议 |
| 🔴 高阶玩法 | 操作要求高，或需要对英雄深度理解 | 对新手降低推荐优先级，仅在没有更好选择时解释 |

以 **循环往复** 为例，完整标签为：

```
循环往复
├─ 效果类型：🔁 技能加速
├─ 适用英雄：通用
├─ 强度评级：A · 强力推荐
├─ 品质等级：金色
├─ 所属套装：无
├─ 装备联动：无特定联动
├─ 对局阶段：全程通用
└─ 决策难度：🟢 简单易懂

教练据此生成推荐话术：
> "【循环往复】万金油选择，技能转得更快，几乎所有英雄都能用，选它不亏。"

以 **虚幻武器** 为例：

```
虚幻武器
├─ 效果类型：🎯 攻击特效
├─ 适用英雄：法术英雄优先 / 物理英雄优先（均可）
├─ 强度评级：B · 看情况（需要配合装备）
├─ 品质等级：金色
├─ 所属套装：无
├─ 装备联动：攻速联动、特定装备（纳什之牙、电刀）

#### B3. 一句话式海克斯解释 + 推荐

设计原则：
- **不给数据**，用自然语言说清"是什么、适合谁、为什么值得选"
- 只在**玩家首次遇到**或**效果不直观**的海克斯上触发
- 推荐优先级：S级（强烈推荐）> 与当前英雄契合 > 通用保底

**话术示例（按符文分类）**：

| 海克斯符文 | 教练一句话 |
|------------|-----------|
| 循环往复 | "万金油选择，让你技能转得更快，几乎所有英雄都能用。" |
| 虚幻武器 | "你的技能会额外触发装备特效，法师出纳什之牙、物理带电刀会特别猛。" |
| 科学狂人 | "天胡符文，碰到就拿，让你变得又硬又能打。" |
| 巨人杀手 | "对面有大肉？选它，越打血厚的伤害越高。" |
| 不详契约 | "用血量换伤害，适合能自保或有治疗的英雄，风险高回报也高。" |
| 坦克引擎 | "选了就是铁墙一块，特别适合前排扛线英雄。" |

- **话术示例**：

> "你已经有一个【大法师】套装的符文了，再选一个同套装的就能激活额外加成，相当于白送一波增强。"

#### C1. 己方英雄简介
- **触发时机**：加载完成 / 等待开局时
- **话术示例**：

> "对面有个机器人，注意躲他的钩子，被钩到基本就没了。躲在小兵后面就行。"

> "对面火男很烦，技能扎堆站就会被烧，散开一点打。"

## 三、触发策略与优先级

### 教练发言优先级（从高到低）

| 优先级 | 内容类型 | 触发条件 |
|--------|----------|----------|
| P0 | 海克斯符文推荐 | 进入强化符文选择界面 |
| P1 | 危险预警 | 敌方关键技能 / 团战即将爆发 |
| P2 | 己方英雄简介 | 加载完成、首次操作 |
| P3 | 敌方英雄预警 | 开局上线期间 |
| P4 | 地图机制引导 | 走路上线、塔被推、首次遇到血包 |
| P5 | 套装组合提醒 | 二次及以上符文选择时 |

### 去重与冷却规则
- 同一条教学内容对同一玩家**仅触发一次**（基于玩家历史记录）
- 同一场对局中，地图机制类提示最多触发 **3 条**
- 教练连续发言间隔不低于 **15 秒**
- 当存在更高优先级内容时，低优先级内容被跳过，不补发

## 四、海克斯选择态完整方案

### 4.0 核心设计原则

海克斯选择是整局中**信息密度最高、决策压力最大**的节点。方案需要在一个界面内同时解决三个问题：

1. **这个符文强不强** → 胜率等级化展示
2. **要不要刷新换一批** → 刷新建议
3. **选哪个最适合我** → 标签 + 教练推荐

以下围绕这三个问题逐一设计。

---

### 4.1 胜率的等级化展示

#### 设计目标
不直接展示胜率数字（避免"数据压迫感"），但让玩家能直观感知每个符文的强度水平。

#### 等级定义

底层数据：取该符文**在当前英雄 × 当前阶段上的胜率**（优先），回退到**当前英雄全阶段胜率**，再回退到**全英雄平均胜率**（兜底）。

| 等级 | 对应胜率区间 | 视觉表现 | 教练话术参考 |
|------|-------------|----------|-------------|
| **S** | ≥ 55% | 金色边框 + 闪光粒子 + "天胡" 角标 | "碰到就拿，别犹豫" |
| **A** | 52% ~ 55% | 金色边框 + 呼吸光效 + "强力" 角标 | "这个很稳，选它不亏" |
| **B** | 49% ~ 52% | 蓝色边框 + "可选" 角标 | "中规中矩，看情况" |
| **C** | 46% ~ 49% | 默认边框，无角标或灰色角标 | "一般般，有更好的可以换" |
| **D** | < 46% | 灰色边框 + 暗淡处理 | "不太推荐，考虑刷新" |

#### 展示方式

每张符文卡片的**右上角**显示等级角标（S/A/B/C/D），样式如下：

```
┌──────────────┐
│          [S] │  ← 等级角标，S 为金色，A 为蓝色，B 为白色，C/D 不显示或灰色
│              │
│   符文图标   │
│   符文名称   │
│              │
│  ──────────  │
│  一句话描述  │
└──────────────┘
```

#### 数据说明
- 等级角标是**对该英雄的相对评级**，不是符文的绝对强度。同一个符文在不同英雄上可能是不同等级。
- 数据来源：后端按 `(英雄ID, 符文ID)` 维度预计算胜率，版本更新时刷新。
- **新符文冷启动**：上线首周数据量不足时，回退到全英雄平均胜率，并标注"新符文"角标替代等级。

### 4.2 刷新建议

#### 机制回顾
在海克斯选择界面，每次三选一时玩家有**1 次刷新机会**，可以刷掉当前 3 个选项换一批新的。刷新后不可再次刷新。

#### 底层数据
对每个符文，后端统计**被刷新率**（Reroll Rate）：

```
被刷新率 = 该符文出现后被玩家刷新的次数 / 该符文出现的总次数
```

这个数据反映了**大量玩家的集体判断**——被刷新率越高，说明大家普遍认为不值得拿。

#### 刷新建议逻辑

综合当前 3 个符文的等级评分，给出整体刷新建议：

| 场景 | 判断条件 | 刷新建议 | UI 表现 |
|------|---------|---------|---------|
| 有天胡 | 3 个中至少 1 个为 S 级 | **不建议刷新** | 刷新按钮置灰 + 提示"已经有很强的了" |
| 都不错 | 3 个中最高为 A 级，且 ≥ 2 个 A/B 级 | **不建议刷新** | 刷新按钮正常 + 提示"这批还行" |
| 一般般 | 3 个中最高为 B 级 | **可以刷新** | 刷新按钮高亮 + 提示"可以试试刷新换一批" |
| 全弱 | 3 个全为 C/D 级 | **建议刷新** | 刷新按钮高亮 + 动效 + 提示"这批比较弱，建议刷一下" |
| 后期审慎 | 第 4 次选择（15 级），且有 ≥ 1 个 A 级 | **不建议刷新** | 提示"最后一次了，稳一点" |

#### 单符文被刷新率的展示

在每张符文卡片上，可以用一个**轻量指示器**暗示该符文的"大众口碑"：

| 被刷新率 | 含义 | 视觉表现 |
|---------|------|---------|
| < 15% | 大家都抢着要 | 符文卡片底部显示 "🔥 热门" 小标签 |
| 15% ~ 40% | 看情况 | 不额外标注 |
| 40% ~ 60% | 偏冷门 | 符文卡片底部显示 "❄️ 冷门" 小标签 |
| > 60% | 大多数人都刷掉了 | 符文卡片底部显示 "⚠️ 常被跳过" 小标签 |

> **设计克制**：这些小标签是辅助信息，不应喧宾夺主。尺寸小、颜色淡，视觉权重低于等级角标和教练推荐。

### 4.3 符文标签体系（面向用户展示）

之前文档中定义的 8 维标签体系主要服务于**后端推荐引擎**。面向用户展示时，需要收敛为**少数几个一目了然的标签**，直接贴在符文卡片上。

#### 用户可见标签分为三类：

##### (a) 强力单卡 —— "这个符文本身就很猛"

| 标签 | 触发条件 | 视觉 | 含义 |
|------|---------|------|------|
| ⭐ 强力单卡 | 等级为 S 或 A，且被刷新率 < 20% | 金色/蓝色标签 | 不需要任何配合，单独就很强 |

判断逻辑：
```
if 符文等级 in [S, A]
   AND 被刷新率 < 20%
   AND 决策难度 = 🟢简单易懂
→ 打上 "⭐ 强力单卡"
```

教练话术：
> "【科学狂人】⭐ 强力单卡——碰到就拿，选了就是又硬又能打。"

##### (b) 黑科技 —— "这个符文和某些东西搭配会炸裂"

黑科技是指**符文 + 符文**、或**符文 + 装备/天赋**形成的非直觉但强力的组合。这类信息对新手价值最大，因为他们很难自己发现。

标签触发：基于预定义的**黑科技组合库**，当候选符文匹配到组合条件时触发。

**黑科技组合库示例**：

| 组合名 | 组成条件 | 核心原理 | 标签展示 |
|--------|---------|---------|---------|
| 不死坦克 | 坦克引擎 + 作弊我能回城 + 心之钢（装备） | 回城叠心之钢无限增长血量，坦克引擎放大收益 | 🧪 黑科技：不死坦克 |
| 无限火力 | 炼狱导管 + 技能流英雄 | 技能击杀刷新所有冷却，连续释放大招 | 🧪 黑科技：无限火力 |
| 死亡轰炸机 | 俯冲轰炸 + 小丑学院 | 主动冲进人堆死亡，触发巨额范围爆炸 | 🧪 黑科技：死亡轰炸机 |
| 雪球快递 | 史上最大雪球 + 利刃华尔兹 | 超远距离雪球开团 + 位移收割 | 🧪 黑科技：雪球快递 |
| 全能吸血鬼 | 吸血习性 + 不祥契约 + 吸血装 | 契约掉的血靠吸血全补回来，净赚伤害加成 | 🧪 黑科技：吸血鬼 |
| 巨人收割机 | 巨人杀手 + 缩小射线 | 缩小降低对方抗性 + 巨人杀手按比例打伤害 | 🧪 黑科技：巨人收割 |
| （持续扩充） | ... | ... | ... |

**触发逻辑**：
```
当前候选符文 A 出现时：
1. 检查玩家已选符文列表中，是否有与 A 构成黑科技的组合
2. 检查玩家当前装备中，是否有与 A 构成黑科技的装备
3. 检查当前 3 个候选符文互相之间，是否构成黑科技
→ 任一命中，符文 A 上打 "🧪 黑科技" 标签
```

教练话术：
> "【作弊我能回城】🧪 配上你已经有的坦克引擎，可以回城叠心之钢无限变肉，这套叫'不死坦克'，非常离谱。"

##### (c) 羁绊关联 —— "这几个符文属于同一套装，凑齐有额外奖励"

指套装体系内的关联。玩家看到的不是"所属套装"这个枯燥标签，而是**当前能凑成什么 + 差几个 + 凑齐得什么**。

**展示逻辑**：

| 场景 | 标签展示 | 教练话术 |
|------|---------|---------|
| 候选符文属于某套装，且玩家还没有该套装的任何符文 | 🔗 俯冲炸弹 ①/④ | "这个符文属于【俯冲炸弹】套装，凑齐能减少复活时间，后面再碰到同套装的可以优先拿。" |
| 候选符文属于某套装，且玩家已有 1 个同套装 | 🔗 俯冲炸弹 ②/④ ⚡可激活 | "你已经有一个【俯冲炸弹】的符文了！再选这个就能激活套装效果，白送一波复活加速。" |
| 候选符文属于某套装，且玩家已有 2+ 个同套装 | 🔗 俯冲炸弹 ③/④ ⚡已激活·可升级 | "再凑一个就满套了，套装效果会更强。" |
| 候选 3 个中有 2 个属于同一套装 | 两张卡片间显示连线 + "🔗 同属XX套装" | "这两个是同一套的，选一个后面再碰到另一个就能凑齐。" |

### 4.6 二次解释机制设计

#### 设计目标

海克斯大乱斗中存在大量"效果描述看起来简单但实际机制复杂"的符文，新手无法通过一句话描述理解其真正含义和使用价值。二次解释机制在不增加信息负担的前提下，为需要深度理解的符文提供"按需深入"的解读入口。

#### 需要二次解释的符文类型

| 类型 | 特征 | 代表符文 | 解释重点 |
|------|------|----------|----------|
| **规则改变型** | 改变大乱斗基本规则 | 作弊我能回城 | 解释"正常大乱斗不能回城"这个前提 |
| **技能替换型** | 替换/改变召唤师技能 | 利刃华尔兹、史上最大雪球 | 解释替换了什么、获得什么、失去什么 |
| **条件触发型** | 需满足特定条件才能触发 | 热身动作、巨人杀手 | 解释触发条件和实际操作方式 |
| **装备联动型** | 效果与特定装备类型绑定 | 头戴式装备大师、暴击专精 | 解释涉及哪些装备、英雄是否通常出这些装备 |
| **反直觉型** | 描述让人误解实际效果 | 不祥契约（以为是debuff） | 纠正常见误解 |
| **黑科技核心型** | 是某个黑科技组合的关键组件 | 坦克引擎+作弊回城 | 解释组合潜力 |

#### 三层递进解释架构

```
┌─── 第一层：白话翻译（What）──────────────────────────────────┐
│  "大乱斗本来不能回城买装备，选了这个就能像普通模式一样按B回城" │
│  → 消除术语壁垒，确保新手理解核心效果                          │
└────────────────────────────────────────────────────────────────┘
         ↓
┌─── 第二层：场景化说明（When & Where）────────────────────────┐
│  "坦克英雄配心之钢最猛，因为可以反复回城叠属性"                │
│  → 建立判断框架：什么情况下好用 / 什么情况下不好用              │
└────────────────────────────────────────────────────────────────┘
         ↓
┌─── 第三层：联动提示（Combo）─────────────────────────────────┐
│  "配合坦克引擎 + 心之钢可组成'不死坦克'黑科技"                │
│  → 揭示隐藏价值：不仅本身好用，还能和其他东西产生化学反应       │
└────────────────────────────────────────────────────────────────┘
```

#### 英雄类型适配解释

同一符文对不同类型英雄的解释侧重不同：

| 符文 | 战士/坦克 | 射手 | 法师 | 刺客 |
|------|-----------|------|------|------|
| 利刃华尔兹 | "多一个突脸手段，E进去就砍" | "慎选，会失去闪现保命" | "不推荐，你不需要冲到人堆里" | "配合技能连招，突进收割利器" |
| 暴击专精 | "不出暴击装的话收益为0" | "必拿！你每件暴击装多10%爆伤" | "完全没用，法师不出暴击装" | "部分刺客走暴击流可选" |
| 作弊我能回城 | "配心之钢无限叠血，坦克天胡" | "可以回城续状态买装备" | "能回城买装备，但没坦克收益大" | "意义不大，刺客不缺续航" |

### 4.7 开局+7级+11级+15级 阶段化推荐策略

#### 设计原则

4 次海克斯选择不是独立的 4 次决策，而是一个**渐进式构建 build 的过程**。教练需要在不同阶段给出差异化的策略指引：

| 阶段 | 核心命题 | 教练角色定位 |
|------|----------|-------------|
| **开局（Lv.1）** | "该拿什么奠基？" | 简单直接，推万金油 |
| **7级** | "该补强还是搞组合？" | 结合已有符文做联动推荐 |
| **11级** | "build 定型了吗？" | 基于已有符文+装备做全局最优推荐 |
| **15级** | "最后的补强点在哪？" | 保守策略，补短板 > 赌组合 |

#### 各阶段详细策略

**开局（Lv.1）**

```
决策优先级：
S 级万金油 > A 级强力单卡 > B 级通用符文 > 刷新

教练策略：
- 黑科技组合：仅提示"后面碰到XX能凑黑科技"，不作为推荐依据
- 套装：仅提示属于哪个套装，不推荐为了凑套降级选择
- 刷新门槛低：全 C/D 直接建议刷新
- 话术风格：简短明快，"开局不纠结，哪个强选哪个"
```

**7级**

```
决策优先级：
S 级 > 黑科技可成型 > A 级单卡 > 差1件套装 > B 级 > 刷新

教练策略：
- 回顾开局选了什么，做联动分析
- 黑科技已具备1个组件 → 优先推荐第2个组件
- 话术带上下文："你开局拿了坦克引擎，现在作弊回城出了，不死坦克要成了！"
- 开始考虑装备方向建议
```

**11级**

```
决策优先级：
S 级 > 黑科技补完 > 与已有装备联动 > 差1件套装升级 > A 级 > B 级

教练策略：
- 分析玩家已有2个符文 + 当前装备的完整 build 状态
- 推荐能最大化已有 build 收益的符文
- 话术示例："你有暴击律动+易损，再拿暴击专精直接三件套，每件暴击装多10%爆伤！"
- 如果已有 build 不连贯，推荐万金油弥补
```

**15级**

```
决策优先级：
S 级 > A 级 > 补短板 > B 级 > 谨慎刷新

教练策略：
- "最后一次了"心理锚定，降低冒险意愿
- 不推荐为了凑套或黑科技选弱符文
- 如果 build 有明显短板（比如全输出无防御），推荐防御/续航符文
- 刷新建议大幅保守化：除非全 D 级否则不建议刷新
- 话术示例："最后一次选择了，A 级很稳，不建议赌刷新"
```

#### "单卡强度 vs 羁绊"决策引导

这是新手最高频的困惑。教练在每个阶段给出明确的决策框架：

```
阶段 × 场景 → 教练建议

开局 + S/A单卡 vs B套装  → "开局选强的，套装后面再说"
7级 + A单卡 vs 差1件套装 → "套装快成了就凑，白送一波增强"
7级 + A单卡 vs 差2件套装 → "套装还远，先选强的"
11级 + 联动已有build的B vs 不联动的A → 比较综合胜率
15级 + 任何 vs 套装       → "最后一次不赌套装，选最稳的"
```

#### 黑科技组合库的维护策略

黑科技组合不能纯靠人工拍脑袋，也不能纯靠数据挖掘。建议两条路并行：

1. **数据挖掘发现**：从大量对局中挖掘"共同出现 + 高胜率"的符文/装备组合
   - 条件：两个符文共同出现胜率 > 单独出现胜率 3% 以上
   - 筛选后人工审核，确认有明确的机制协同而非纯巧合
2. **人工策划补充**：版本更新时，策划团队/高段位玩家主动补充新发现的黑科技
3. **社区反馈**：从攻略社区、直播、短视频中收集热门combo，验证数据后入库

## 五、教练语言风格规范

| 维度 | 要求 |
|------|------|
| 语气 | 轻松、像朋友聊天，不说教 |
| 长度 | 单条不超过 30 字（符文推荐可放宽至 40 字） |
| 表达 | 不给数值和百分比，用感受性描述（"很猛""特别适合""白送"） |
| 频率 | 宁可少说，不要刷屏；宁可晚说，不要抢关键操作时机 |
| 人称 | 用"你"而非"玩家""召唤师"，保持亲近感 |
| 新手友好 | 避免术语缩写（不说"CD"说"技能冷却"、不说"AD"说"物理攻击"），首次出现游戏术语时带一句白话解释 |

## 六、效果度量建议

| 指标 | 说明 |
|------|------|
| 海斗模式进入率 | 新手在教练引导下主动进入海斗的比例变化 |
| 对局完成率 | 新手是否更少中途退出 |
| 次日 / 7日留存 | 核心留存指标 |
| 符文选择正确率 | 新手选择的符文是否更趋近于高胜率推荐 |
| 教练互动率 | 推荐面板的展开/查看比例 |
| 教练满意度 | 局后问卷或点赞/踩机制 |

## 七、分期建议

### Phase 1（MVP）
- 对局开始时的英雄简介 + 地图机制穿插提示
- 海克斯选择态专属 UI + 一句话符文推荐

- 教练个性化：语音风格可选（热血解说 / 温柔姐姐 / 毒舌损友）
