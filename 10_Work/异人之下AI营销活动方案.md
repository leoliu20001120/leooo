**版本**: v0.1 | **日期**: 2026-03-20  
**作者**: AI 产品 × 营销策略  
**背景**: 游戏预约量破 2500 万，2026 下半年公测，王也 NPC 知识库 1125+ 条已就绪

---

## 1. 核心洞察

### 为什么要做 AI 营销？

| 维度 | 分析 |
|------|------|
| **市场时机** | 和平精英 AI NPC 累计体验 1.1 亿用户、逆水寒沈秋索上线 2 小时千万人涌入——AI NPC 已被验证为流量引爆器 |
| **IP 优势** | 一人之下漫画阅读量 319.2 亿、动画豆瓣 9+、第六季热播——IP 热度正在高点 |
| **产品优势** | 王也 NPC 知识库 1125+ 条，28 种场景回复模板，甲申之乱 23 条详解——已具备最深的 AI 角色"灵魂" |
| **受众特征** | 20-29 岁男性为主（约 60%+80%），有付费能力，爱发声、爱讨论，天然适合 UGC 传播 |
| **竞品空白** | 当前 AI NPC 营销以 MMO/射击为主，国漫 IP + 格斗 + AI NPC 的组合尚无先例 |

### 一句话定位

> **"公测前，让 2500 万预约玩家先跟王也聊起来"**

## 4. 第一阶段：认识王也（预热期 — 公测前 2 个月）

### 4.1 活动：「问道王也」AI 对话体验

**核心玩法**：开放王也 AI NPC 对话入口，玩家可以在公测前先跟王也聊天

| 项目 | 设计 |
|------|------|
| **入口** | 官网/微信小程序/TapTap 内嵌 H5 页面 |
| **体验形式** | 聊天界面 + 王也立绘 + 语音（可选 TTS） |
| **对话能力** | 基于 1125+ 条知识库 → 7 大场景（自我介绍/日常/修行/战斗/友情/建议/哲学幽默） |
| **IP 彩蛋** | 输入特定关键词（如"甲申之乱""风后奇门""诸葛青"）触发隐藏深度回复 |
| **社交钩子** | 每次对话结束生成「问道签」（随机王也金句 + 用户名 + 精美卡面），一键分享朋友圈 |

### 4.2 活动：「异人鉴定局」AI 人格测试

**核心玩法**：与王也 AI 进行 5 轮对话，AI 根据你的回答判断你适合哪个门派/组织

| 项目       | 设计                            |
| -------- | ----------------------------- |
| **流程**   | 5 轮对话式问答（非传统问卷，是 AI 动态追问）     |
| **结果**   | 12 种结果（天师府/全性/武当/临时工等 12 个组织） |
| **产出**   | 生成个性化「异人身份证」（门派+异能属性+王也评语）    |
| **裂变**   | "看看你朋友是什么异人" → 分享后好友点击直接进入测试  |
| **数据价值** | 收集用户偏好数据 → 公测后做个性化推送          |

### KPI

| 指标 | 目标 |
|------|------|
| AI 对话总参与人数 | 300 万+ |
| 人均对话轮次 | ≥8 轮 |
| 「问道签」分享率 | ≥25% |
| 「异人身份证」裂变系数 | ≥1.5（每人带来 1.5 个新用户） |
| 新增预约数 | 200 万+ |

## 5. 第二阶段：挑战王也（爆发期 — 公测前 2 周至公测）

### 5.1 活动：「论道天下」AI 辩论挑战赛

**核心玩法**：玩家选择一个话题跟王也 AI 辩论，AI 根据辩论表现打分

| 项目         | 设计                                           |
| ---------- | -------------------------------------------- |
| **话题池**    | 10 个一人之下世界观话题（"全性的理念对不对""异人该不该入世""甲申之乱谁是正义"） |
| **辩论形式**   | 3 轮攻防，每轮限时 60 秒输入                            |
| **AI 评判**  | 王也以道家视角回应 + 给出"说服力评分"（S/A/B/C 四档）            |
| **排行榜**    | 全服排行 → 前 100 名获得公测限定称号「舌战群儒」                 |
| **UGC 爆点** | 鼓励玩家录屏/截图分享辩论内容 → 天然的内容素材                    |

### 5.2 活动：「入世之约」AI 角色互动剧情

**核心玩法**：王也 AI 通过对话引导玩家体验一段原创互动剧情（入世篇前传）

| 项目         | 设计                                |
| ---------- | --------------------------------- |
| **形式**     | 类似"文字冒险游戏"，AI 实时生成剧情分支            |
| **剧情线**    | 王也带你夜探天下会 → 遭遇异人 → 根据你的选择走向不同结局   |
| **结局数**    | 5 个结局（正义/黑化/隐居/觉醒/彩蛋）             |
| **奖励**     | 集齐 5 个结局 → 解锁公测限定「五行轮转」头像框        |
| **KOL 联动** | 邀请 B站/抖音 UP主直播挑战全结局，带话题 #异人之下AI王也 |

### 5.3 公测倒计时：王也的每日一言

**形式**：公测前 14 天，每天 AI 生成一条王也风格的「倒计时金句」，自动发布到全渠道

```
Day 14: "入世嘛，急什么。先把心静下来。" — 王也
Day 7:  "七天？正好够练一套风后奇门。" — 王也  
Day 1:  "明天见。记住，拳头硬的才有道理。" — 王也
```

### KPI

| 指标 | 目标 |
|------|------|
| 辩论赛参与人数 | 100 万+ |
| 互动剧情完成率 | ≥60%（至少打出 1 个结局） |
| UGC 内容产出量 | 10 万条+（含截图/录屏/二创） |
| #异人之下AI王也 话题阅读量 | 5 亿+ |
| 公测首日下载量 | 目标值由运营定 |

## 6. 第三阶段：活在王也身边（长尾期 — 公测后持续）

### 6.1 游戏内 AI NPC 自然延续

- 预热期积累的对话数据 → 反哺游戏内王也 NPC 表现
- 玩家在预热期的选择（门派/结局）→ 影响游戏内王也对你的态度（个性化记忆）

### 6.2 「道友圈」AI 社交功能

- 玩家可以随时在游戏内/小程序中跟王也闲聊
- 王也根据玩家游戏进度给出个性化建议（"你最近打了不少架，要不要修炼修炼？"）
- 节日/版本更新时，王也主动发起话题

### 6.3 持续运营：AI 角色宇宙扩展

| 时间 | 新角色 | 定位 |
|------|--------|------|
| 公测 +1 月 | 冯宝宝 AI | 冷面反差萌 |
| 公测 +2 月 | 张楚岚 AI | 嘴贱搞笑担当 |
| 公测 +3 月 | 诸葛青 AI | 毒舌吐槽 |

## 9. 风险与对策

| 风险 | 概率 | 影响 | 对策 |

## 12. 核心创新点总结

1. **"先聊后玩"** — 打破"公测才能体验"的惯例，用 AI 对话让玩家提前建立角色情感连接
2. **AI 驱动的 UGC** — 每个玩家和王也的对话都是独一无二的内容素材，天然适合传播
3. **预热数据反哺正式服** — 玩家的预热期选择影响游戏内 NPC 对你的态度，创造"养成感"
4. **IP 知识库深度 = 护城河** — 1125+ 条知识库 + 28 种场景回复 = 市面上最"有灵魂"的 AI 角色
