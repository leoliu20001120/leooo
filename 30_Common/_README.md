# 📚 30_Common — 通识知识沉淀

> 跨项目的通用知识、方法论、分析框架，长期积累的"武器库"。

## 目录结构

```
30_Common/
├── 方法论/               # 分析框架、思维模型、工作方法
│   ├── MECE分析法.md
│   ├── 数据分析框架.md
│   └── ...
└── 行业知识/             # 领域知识积累
    ├── 游戏行业/
    └── ...
```

## 什么内容放这里？

从 `10_Work`（工作）和 `20_Study`（学习）中**提炼出的可复用知识**：

| 来源 | 提炼后放这里 |
|------|-------------|
| 多个工作项目都用到的分析方法 | `方法论/` |
| 学习中总结的通用技巧 | `方法论/` |
| 行业趋势、市场洞察 | `行业知识/` |
| 反复查阅的参考资料 | 对应子目录 |

## 使用场景

- AI 回答问题时可以参考此目录下的内容作为上下文
- 写新笔记时用 `[[双链]]` 关联到这里的知识卡片
- 定期从工作和学习中提炼"可复用经验"沉淀到此处

## 命名建议

```
主题_简短描述.md
```

## AI 约定

- AI 可以主动建议将重复出现的知识点沉淀到此目录
- 新建笔记时检查是否已有相关的通识知识可以关联
