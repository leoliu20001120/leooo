## CaloriSnap

### 项目信息
- **类型**: Side Project | **路径**: `40_Projects/CaloriSnap/`
- **定位**: 拍照识别奶茶/咖啡出杯标签 → 计算热量 + 咖啡因
- **技术**: 微信小程序 + FastAPI + Supabase(PostgreSQL) + OCR/LLM

### 版本演进
| 版本 | 日期 | 关键变化 |
|------|------|---------|
| v0.1 | 03-18 | 基础原型，6页面 |
| v0.2 | 03-18 | +咖啡因追踪，双色系统 |
| v0.2.1 | 03-18 | 配色升级「茶咖双生」 |
| v0.3 | 03-18 | 第一性原理Review，信息架构重构，+三态 |

### 数据库
- 7品牌 160+SKU，35+小料，8表+2视图+RLS

### 设计决策
- 配色: 焦糖棕`#C47A4A`(热量) + 翡翠绿`#2D8C6F`(咖啡因)
- CTA在首页最显眼，水平进度条替代环形仪表盘
- 待办: Phase 0 技术验证（OCR方案选型）→ MVP
