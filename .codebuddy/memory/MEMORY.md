## 关键教训（快速回忆）

1. macOS定时任务 → **永远用launchd**，crontab睡眠不执行
2. UI设计 → **功能性 > 装饰性**，问自己"服务用户还是设计师审美？"
3. 状态覆盖 → 每个功能设计 **正常/空/错误** 三种状态
4. 新数据入库 → **先交叉校验**，标注一致/冲突/新增
5. CTA → 首页最显眼位置，核心任务为中心
