# 📚 20_Common — 通识知识沉淀

> 跨项目的通用知识、方法论、分析框架，长期积累的"武器库"。

## 推荐子目录

```
20_Common/
├── 方法论/             # 分析框架、思维模型
├── 技术栈/             # 编程语言、工具使用笔记
├── 行业知识/           # 领域知识积累
└── 读书笔记/           # 书摘和阅读总结
```

## 使用场景

- 从项目中提炼出的**可复用知识**放在这里
- AI 回答问题时可以参考此目录下的内容作为上下文
- 写新笔记时用 `[[双链]]` 关联到这里的知识卡片

## 命名建议

主题_简短描述.md
