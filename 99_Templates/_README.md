# 🧩 99_Templates — 模板库

> Obsidian 模板文件，搭配核心插件 "Templates" 或社区插件 "Templater" 使用。

## 使用方式

1. Obsidian 设置 → 核心插件 → 启用 **模板**
2. 设置模板文件夹为 `99_Templates`
3. 新建笔记时按 `Ctrl/Cmd + T` 插入模板

## 模板列表

| 模板文件 | 用途 |
|---------|------|
| `tpl_daily_note.md` | 每日笔记 |
| `tpl_project.md` | 新项目启动 |
| `tpl_meeting.md` | 会议记录 |
