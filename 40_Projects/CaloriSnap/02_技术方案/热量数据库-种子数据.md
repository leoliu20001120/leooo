## 📐 糖度热量换算公式（通用）

> 不同品牌的糖度叫法不同，但热量增量大致可用以下公式估算

| 糖度 | 别名 | 额外热量(约) | 额外糖量(约) |
|------|------|------------|-------------|
| 无糖/去糖 | 0分甜 | 基准 | 0g |
| 微糖 | 一分甜/少少少甜 | +25-40 kcal | +6-10g |
| 少糖 | 三分甜/少少甜 | +50-80 kcal | +12-20g |
| 半糖 | 五分甜/少甜 | +70-120 kcal | +18-30g |
| 正常糖 | 七分甜/多甜 | +85-160 kcal | +21-40g |
| 全糖 | 十分甜 | +100-200 kcal | +25-50g |

### 杯型热量换算

| 杯型变化 | 热量系数 |
|---------|---------|
| 中杯 → 大杯 | × 1.25~1.35 |
| 大杯 → 超大杯 | × 1.15~1.25 |
| 热饮 → 冰饮 | × 0.85~0.95 |

## 🏷️ 出杯标签格式分析（OCR 设计参考）

### 标签物理特征

| 属性 | 说明 |
|------|------|
| 材质 | 热敏不干胶纸 |
| 尺寸 | 常见 40×30mm 或 60×40mm |
| 打印 | 热敏打印，黑白 |
| 位置 | 贴在杯身侧面或杯盖上 |

### 标签包含的典型字段

| 字段 | 出现频率 | 示例 | OCR难度 |
|------|---------|------|---------|
| **品牌名/Logo** | 常见 | 瑞幸咖啡、喜茶 | 🟢 低 |
| **饮品名** | 必有 | 生椰拿铁、伯牙绝弦 | 🟢 低 |
| **杯型/规格** | 必有 | 大杯/中杯、Grande | 🟢 低 |
| **温度** | 常见 | 冰/热/去冰/少冰 | 🟢 低 |
| **糖度/甜度** | 常见 | 半糖/少甜/标准糖 | 🟢 低 |
| **加料/小料** | 有加料时 | +珍珠 +椰果 +芝士 | 🟡 中 |
| **序号/单号** | 常见 | #086 | 🟢 低 |
| **时间** | 常见 | 14:35 | 🟢 低 |
| **备注** | 偶尔 | 少冰多料 | 🟡 中 |

### OCR → 热量计算 的关键字段提取

```
从标签 OCR 文字中需要提取：

1. 品牌（brand）— 通过 Logo 或品牌名识别
2. 品名（drink_name）— 核心匹配字段
3. 杯型（size）— 影响热量系数
4. 糖度（sugar_level）— 影响额外热量
5. 加料（toppings[]）— 每种加料叠加热量
6. 温度（temperature）— 冰/热影响热量

热量计算公式：
total = base_calories × size_factor × temp_factor + sugar_extra + Σ(topping_calories)
```
