# 🎨 CaloriSnap 产品原型说明 v0.2

> 原型文件: [prototype.html](prototype.html)
> 打开方式: 浏览器直接打开，建议用 Chrome 开发者工具模拟手机尺寸 (iPhone 14 Pro, 390×844)
> 版本: v0.2 — 新增咖啡因追踪 + UI 全面升级

## 🆕 v0.2 更新摘要

| 升级项 | v0.1 | v0.2 |
|--------|------|------|
| 健康指标 | 仅热量 | **热量 + 咖啡因** 双轴追踪 |
| 首页设计 | 单一热量卡片 | **双环形仪表盘**（热量🔥 / 咖啡因⚡） |
| 字体 | 系统字体 | **DM Sans + Noto Sans SC** 混排 |
| 视觉风格 | 标准卡片式 | 磨砂玻璃 + 径向渐变背景 + 微动画 |
| 等价物 | 3个固定卡片 | 横向滚动 Chip 条 + 咖啡因等价物 |
| 健康提示 | 无 | **智能咖啡因提示**（基于时间 + 摄入量） |
| 结果页 | 单一热量数字 | **双指标大数字** + 咖啡因明细 |
| 报告页 | 单柱状图 | **双色柱状图**（热量 + 咖啡因对比） |
| 设置页 | 仅热量预算 | 热量预算 + **咖啡因限额** + **宵禁时间** |
| 拍照页 | 虚线框 | **四角标记 + 扫描线动画** |
| 识别动画 | 单环旋转 | **双轨道旋转**（橙+青双色） |

## 📱 页面结构

### 1️⃣ 首页 (Home) — 双轴健康仪表盘

```
┌─────────────────────────────┐
│ ☀️ 下午好，健康达人           │ ← 渐变橙色 Hero
│ 今日健康报告                  │
├──────────┬──────────────────┤
│  🔥 热量   │  ⚡ 咖啡因       │
│  ╭───╮    │   ╭───╮         │ ← SVG 环形仪表盘
│  │347│    │   │195│         │
│  │kcal│   │   │ mg│         │
│  ╰───╯    │   ╰───╯         │
│  /500  剩153│  /400  剩205   │
├──────────┴──────────────────┤
│ 🍚1.5碗 🏃58分 ☕≈2.6杯 😴17:30│ ← 横滚 Chip 等价物
├─────────────────────────────┤
│ ☕ 咖啡因提示                  │ ← 智能健康提示卡
│ 已摄入195mg，建议17:30前停止   │
├─────────────────────────────┤
│ 今日记录                      │
│ ☕ 生椰拿铁   217kcal ⚡150mg │ ← 双指标记录卡
```

**设计亮点：**
- **双环形仪表盘**：左橙（热量）右青（咖啡因），SVG 绘制，加载时有动画
- **智能提示**：基于当前咖啡因摄入量和时间，动态提示停止饮用时间
- **Chip 等价物**：新增「☕ ≈ X 杯浓缩」和「😴 最晚饮用时间」

## 🎨 设计规范 v0.2

### 配色方案 — 双色系统

| 颜色 | 色值 | 用途 |
|------|------|------|
| **热量橙** | `#FF6B35` | 热量指标、品牌主色、CTA |
| 热量橙浅 | `#FFEFE7` | 热量卡背景 |
| **咖啡因青** | `#0ABEAD` | 咖啡因指标、健康提示 |
| 咖啡因青浅 | `#E0FAF7` | 咖啡因卡背景、Badge |
| 安全绿 | `#22C55E` | 达标、低风险 |
| 警示黄 | `#FBBF24` | 中等、接近阈值 |
| 危险红 | `#EF4444` | 超标、高风险 |

### 字体

| 用途 | 字体 | 大小/字重 |
|------|------|-----------|
| 英文/数字 Display | DM Sans | 26-56px / 800 |
| 中文正文 | Noto Sans SC | 13-17px / 400-700 |
| 仪表盘数字 | DM Sans | 26px / 800 |
| 辅助文字 | Noto Sans SC | 10-12px / 400-500 |

### 视觉效果

- **磨砂玻璃 Tab Bar**: `backdrop-filter: blur(20px)` + 半透明白
- **径向渐变背景**: 橙色 + 青色双光源
- **环形进度动画**: SVG stroke-dashoffset 过渡 1.2s
- **页面切换动画**: fadeUp 0.35s（上移淡入）
- **扫描线**: 橙色渐变光带，2.5s 循环
- **双轨道旋转**: 外圈正转 1.2s / 内圈反转 1.8s

### 圆角系统

| 级别 | 大小 | 用途 |
|------|------|------|
| SM | 12px | 卡片、按钮 |
| MD | 18px | 大卡片、详情 |
| LG | 24px | 弹窗、面板 |
| XL | 32px | Hero 区域底部 |

## 🔄 交互流程 v0.2

```
首页（双仪表盘）──→ 点击 📷 ──→ 拍照页
                                    │
                              点击快门
                                    │
                          双轨道识别动画(3s)
                                    │
                              结果页（热量 + 咖啡因）
                               │         │
                         [再拍一杯]  [记入日志]
                           │              │
                        拍照页         首页（更新双仪表盘）
```

### 智能健康提示逻辑

```
IF 当前时间 > 咖啡因宵禁时间:
  → "现在已超过宵禁时间，不建议再摄入含咖啡因饮品"

ELIF 今日咖啡因 > 300mg:
  → "今日咖啡因摄入较高(Xmg)，建议选择无咖啡因饮品"

ELIF 今日咖啡因 > 200mg:
  → "已摄入Xmg咖啡因，建议HH:MM前不再摄入"

ELSE:
  → "咖啡因摄入适中，注意HH:MM后不要喝含咖啡因饮品"
```
