## 📋 项目管理约定

### 每个项目目录结构

```
项目名/
├── _README.md         ← 项目概览与进度
├── 01_产品设计/        ← PRD、用户故事、原型
├── 02_技术方案/        ← 架构设计、技术选型
├── 03_开发记录/        ← 开发日志、踩坑记录
└── 04_运营数据/        ← 上线后的数据跟踪
```

### 项目状态

- 🔵 **想法** — 只有一个概念
- 🟡 **规划中** — 正在做产品设计和技术选型
- 🟢 **开发中** — 已进入开发阶段
- 🟣 **已上线** — MVP 已发布
- ⚫ **搁置** — 暂停开发
